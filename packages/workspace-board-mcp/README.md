# @webcli/workspace-board-mcp

MCP (stdio) server for workspace kanban boards. Bundled with WebCLI.

Storage: `<workspace>/.webcli/board.json` (same file as the WebCLI Board UI).

Built automatically by `npm run build` from the repo root. Default `~/.webcli/mcp.json` points at:

`packages/workspace-board-mcp/dist/index.js`
